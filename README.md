 Machine Learning Lab 1 - Group Assignment
Problem Description
This project analyzes open data related to Umrah companies and agencies for the 1446H season in Saudi Arabia. 
The objective is to explore the dataset structure and analyze available company information.

Dataset Source
The dataset was obtained from the Saudi Government Open Data Portal (SDAIA). 
It contains registered Umrah companies and agencies information for the 1446H season.

 Dataset Description
The dataset includes company names, company email addresses, agency names, agency emails, and country information. 
The data is structured in tabular format and will be analyzed using Python and Pandas.

Tools Used
- Python
- Pandas
- Google Colab
