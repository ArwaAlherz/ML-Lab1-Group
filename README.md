# ML-Lab-Group

Machine Learning Lab 1 - Group Assignment

## Problem Description

This project analyzes open data related to Umrah companies and agencies in Saudi Arabia using machine learning techniques. The project focuses on preprocessing, exploratory data analysis (EDA), K-Means Clustering, and Decision Tree classification to discover hidden patterns and analyze categorical data related to Umrah services.

## Dataset Source

The dataset was obtained from the Saudi Government Open Data Portal (SDAIA). It contains registered Umrah companies and agencies information for the 1446H season in Saudi Arabia.

## Dataset Description

The dataset includes company names, company email addresses, agency names, agency emails, and country information. The data was preprocessed and converted into numerical format using Label Encoding techniques before applying machine learning algorithms.

## Techniques Used

- Exploratory Data Analysis (EDA)
- Label Encoding
- K-Means Clustering
- Decision Tree Classification

## Tools Used

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter Notebook / Google Colab
